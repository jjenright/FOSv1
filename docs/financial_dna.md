# Financial DNA

The Financial DNA view is produced from the combined `Annual_KPIs`,
`Spending_Evolution`, `Insights`, and `Action_Plan` sheets.

It describes the household pattern across:

- income and known spending evolution,
- fixed and variable cost pressure,
- liquidity and debt resilience,
- wealth-building consistency,
- major mapped category movements, and
- data-quality limitations.

No private financial values are stored in this repository document. The current
Financial DNA is calculated only when `src.update` runs against the private source
workbook, and the results remain in the excluded `output\` directory.
