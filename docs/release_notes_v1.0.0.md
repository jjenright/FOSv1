# FOS v1.0.0 Release Notes

v1.0.0 completes the Family Financial Operating System planned roadmap.

## Production hardening

- Centralized the release version in the root `VERSION` file.
- Replaced hardcoded private-workbook totals with dynamic reconciliation checks.
- Added generated-XLSX archive and table-integrity validation.
- Added a Windows one-command launcher.
- Added static release-contract checks and automated tests.
- Completed architecture, user, privacy, backup and maintenance documentation.

## Preserved calculations

The v0.6.0 category dictionary, historical import, KPI formulas, dashboard,
insight thresholds and action-generation logic are unchanged.
